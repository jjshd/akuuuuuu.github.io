Outputs
The API offers various ways to receive your data for easy parsing. The output choice will go AFTER the API url. Currently supported outputs are:
Nothing If you do not specify a format, it will be outputted as a basic string.
GET
JSON Your data will be outputted with a JSON format, and it will usually include status codes.
GET
RTVars Will use the same format that Growtopia uses.
